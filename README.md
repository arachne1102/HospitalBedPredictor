# Hospital Bed Predictor

이 프로젝트는 병원 병상 수요를 예측하여 의료 자원의 효율적인 배분을 돕는 것을 목표로 합니다. 현재 데이터 수집 코드를 작성한 상태이며, 프로젝트 진행 상황에 따라 문서가 업데이트될 예정입니다.
